Математический метод
====================

Постановка задачи
-----------------

Библиотека работает с однородной линейной автономной системой с постоянными
коэффициентами:

.. math::

   \frac{dx}{dt} = A x,
   \qquad x = (x_1, \ldots, x_n)^T \in \mathbb{R}^{n},
   \qquad A \in \mathbb{R}^{n \times n}.

Цель — построить автономные первые интегралы, то есть скалярные функции
``W(x)``, постоянные на траекториях системы.

В реализации используется транспонированная матрица

.. math::

   B = A^T.

Если ``v`` — собственный вектор матрицы ``B`` с собственным числом ``lambda``,
то линейная форма

.. math::

   p_v(x) = v^T x

удовлетворяет равенству

.. math::

   \frac{d}{dt} p_v(x(t))
   = v^T A x(t)
   = (A^T v)^T x(t)
   = \lambda p_v(x(t)).

Значит, ``p_v`` является частным интегралом. Из таких частных интегралов метод
строит настоящие первые интегралы: отношения, произведения степеней,
экспоненциальные выражения и угловые комбинации через ``arctan``.

Критерий через производную Ли
-----------------------------

Для скалярного выражения ``W(x)`` производная Ли вдоль системы ``dot{x} = A x``
равна

.. math::

   \mathcal{L}_{A} W(x)
   =
   \sum_{i=1}^{n}
   \frac{\partial W}{\partial x_i}(Ax)_i.

Функция ``W`` является первым интегралом на области, где она определена, если

.. math::

   \mathcal{L}_{A} W(x) = 0.

Все тесты библиотеки проверяют именно это тождество средствами Sage.

Формулы для вещественных собственных чисел
------------------------------------------

Нулевое вещественное собственное число
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Если

.. math::

   Bv = 0,

то

.. math::

   W(x) = p_v(x)

является первым интегралом. В коде такой интеграл имеет ``kind = "real_zero"``.

Одинаковые ненулевые вещественные собственные числа
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Если

.. math::

   Bv_1 = \lambda v_1,
   \qquad
   Bv_2 = \lambda v_2,
   \qquad
   \lambda \ne 0,

то отношение

.. math::

   W(x) = \frac{p_{v_2}(x)}{p_{v_1}(x)}

является первым интегралом на области ``p_v1(x) != 0``. В коде такой интеграл
имеет ``kind = "real_ratio"``.

Разные ненулевые вещественные собственные числа
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Пусть

.. math::

   B v_1 = \lambda_1 v_1,
   \qquad
   B v_2 = \lambda_2 v_2,
   \qquad
   \lambda_1 \ne \lambda_2,
   \qquad
   \lambda_1 \lambda_2 \ne 0.

Статья задает показатели ``h1`` и ``h2`` из условия

.. math::

   \lambda_1 h_1 + \lambda_2 h_2 = 0.

В реализации используется конкретный выбор

.. math::

   h_1 = \lambda_2,
   \qquad
   h_2 = -\lambda_1.

Получается первый интеграл

.. math::

   W(x) = p_{v_1}(x)^{\lambda_2} p_{v_2}(x)^{-\lambda_1}.

Он имеет ``kind = "real_pair"``. В статье аналогичная формула записана через
модули. Библиотека возвращает символьное Sage-выражение и сохраняет
соответствующие ограничения области в ``domain_conditions``.

Формулы для комплексных собственных чисел
-----------------------------------------

Пусть

.. math::

   \lambda = \xi + \zeta i,
   \qquad
   \zeta \ne 0,
   \qquad
   v = \eta + \mu i,

где ``eta`` и ``mu`` — вещественные векторы. Введем линейные формы

.. math::

   p_\eta(x) = \eta^T x,
   \qquad
   p_\mu(x) = \mu^T x.

Одно комплексное собственное число
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Первый интеграл, построенный по одному комплексному собственному числу:

.. math::

   W(x)
   =
   \left(p_\eta(x)^2 + p_\mu(x)^2\right)
   \exp\left(
     -2\frac{\xi}{\zeta}
     \arctan\frac{p_\mu(x)}{p_\eta(x)}
   \right).

В коде он имеет ``kind = "complex_eigen"``. Область определения зависит от
выбранной ветви ``arctan``; в частности, выражение использует отношение
``p_mu(x) / p_eta(x)``, поэтому ``p_eta(x)`` фиксируется как условие области.

Комплексное и вещественное собственные числа
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Пусть есть комплексное собственное число ``lambda = xi + zeta*i`` с вектором
``eta + mu*i`` и ненулевое вещественное собственное число ``lambda_r`` с
вещественным собственным вектором ``v_r``. Тогда строится интеграл

.. math::

   W(x)
   =
   p_{v_r}(x)
   \exp\left(
     -\frac{\lambda_r}{\zeta}
     \arctan\frac{p_\mu(x)}{p_\eta(x)}
   \right).

В коде он имеет ``kind = "complex_real"``.

Два комплексных собственных числа
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Для двух комплексных собственных чисел

.. math::

   \lambda_1 = \xi_1 + \zeta_1 i,
   \qquad
   \lambda_2 = \xi_2 + \zeta_2 i

с собственными векторами ``eta_1 + mu_1*i`` и ``eta_2 + mu_2*i`` строится
угловой первый интеграл

.. math::

   W(x)
   =
   \zeta_1 \arctan\frac{p_{\mu_2}(x)}{p_{\eta_2}(x)}
   -
   \zeta_2 \arctan\frac{p_{\mu_1}(x)}{p_{\eta_1}(x)}.

В коде он имеет ``kind = "complex_pair"``.

Соглашения для жордановых цепочек
---------------------------------

Для жордановой цепочки матрицы ``B`` с собственным числом ``lambda`` библиотека
использует нормировку из статьи:

.. math::

   (B - \lambda E)v^k = k v^{k-1},
   \qquad k = 1, \ldots, m-1.

Sage возвращает обычную жорданову цепочку с единицами на наддиагонали. Поэтому
реализация умножает ``k``-й вектор цепочки на ``k!``, чтобы привести его к
нормировке статьи.

Для каждого вектора цепочки определим

.. math::

   p_k(x) = (v^k)^T x.

Вещественная жорданова цепочка
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Если собственное число вещественное, ненулевое, и длина цепочки хотя бы 2, то
строится интеграл

.. math::

   W(x)
   =
   p_0(x)
   \exp\left(-\lambda \frac{p_1(x)}{p_0(x)}\right).

В коде он имеет ``kind = "real_jordan_exponential"``.

Дополнительные инварианты цепочки задаются функциями ``Psi_i`` из системы

.. math::

   p_k(x)
   =
   \sum_{i=1}^{k}
   \binom{k-1}{i-1}
   \Psi_i(x) p_{k-i}(x),
   \qquad k = 1, \ldots, m-1.

Реализация решает эту систему рекурсивно:

.. math::

   \Psi_k(x)
   =
   \frac{
     p_k(x)
     -
     \sum_{i=1}^{k-1}
     \binom{k-1}{i-1}\Psi_i(x)p_{k-i}(x)
   }{p_0(x)}.

Для ``k >= 2`` выражение ``Psi_k`` возвращается как первый интеграл с
``kind = "real_jordan_psi_k"``. Например,

.. math::

   \Psi_2(x) = \frac{p_0(x)p_2(x) - p_1(x)^2}{p_0(x)^2}.

Комплексная жорданова цепочка
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Пусть

.. math::

   \lambda = \xi + \zeta i,
   \qquad
   v^0 = \eta^0 + \mu^0 i,
   \qquad
   v^1 = \eta^1 + \mu^1 i.

Введем обозначения

.. math::

   r_0(x)^2 = p_{\eta^0}(x)^2 + p_{\mu^0}(x)^2,

.. math::

   d(x) = p_{\eta^0}(x)p_{\eta^1}(x) + p_{\mu^0}(x)p_{\mu^1}(x),

.. math::

   c(x) = p_{\eta^0}(x)p_{\mu^1}(x) - p_{\mu^0}(x)p_{\eta^1}(x).

Экспоненциальный интеграл цепочки:

.. math::

   W_1(x)
   =
   r_0(x)^2
   \exp\left(
     -2\frac{\xi d(x) - \zeta c(x)}{r_0(x)^2}
   \right).

В коде он имеет ``kind = "complex_jordan_exponential"``.

Угловой интеграл цепочки:

.. math::

   W_2(x)
   =
   \arctan\frac{p_{\mu^0}(x)}{p_{\eta^0}(x)}
   -
   \frac{\zeta d(x) + \xi c(x)}{r_0(x)^2}.

В коде он имеет ``kind = "complex_jordan_angle"``.

Для более длинных комплексных цепочек ``Psi_k`` вычисляются над комплексным
полем. Их вещественная и мнимая части возвращаются как
``complex_jordan_psi_k_real`` и ``complex_jordan_psi_k_imaginary``.

Нулевая жорданова цепочка и другое собственное число
----------------------------------------------------

Если нулевому собственному числу соответствует цепочка ``v^0, v^1``, а также
есть ненулевое вещественное собственное число ``lambda_r`` с собственным
вектором ``v_r``, то строится интеграл

.. math::

   W(x)
   =
   p_{v_r}(x)
   \exp\left(-\lambda_r\frac{p_1(x)}{p_0(x)}\right).

В коде он имеет ``kind = "zero_jordan_real"``.

Если другое собственное число комплексное, ``lambda = xi + zeta*i``, с вектором
``eta + mu*i``, то строятся два интеграла:

.. math::

   W_1(x)
   =
   \left(p_\eta(x)^2 + p_\mu(x)^2\right)
   \exp\left(-2\xi\frac{p_1(x)}{p_0(x)}\right),

и

.. math::

   W_2(x)
   =
   \arctan\frac{p_\mu(x)}{p_\eta(x)}
   -
   \zeta\frac{p_1(x)}{p_0(x)}.

В коде они имеют ``kind = "zero_jordan_complex_radius"`` и
``kind = "zero_jordan_complex_angle"``.
